# Module 5 – NLP Applications

This module demonstrates practical Natural Language Processing (NLP) tasks using pretrained Transformer models from the Hugging Face ecosystem.

Each notebook focuses on a specific NLP application, illustrating the complete workflow from preprocessing and model loading to inference or fine-tuning.

---

## Topics Covered

- Token Classification
- Masked Language Modeling
- Machine Translation
- Text Summarization
- Causal Language Modeling

---

## Notebooks

| Notebook | Description |
|----------|-------------|
| Token Classification | Named Entity Recognition and token-level prediction |
| Masked Language Modeling | Fine-tuning BERT-style masked language models |
| Machine Translation | Neural machine translation with encoder-decoder models |
| Text Summarization | Abstractive summarization using Transformer models |
| Causal Language Modeling | Training autoregressive language models from scratch |
